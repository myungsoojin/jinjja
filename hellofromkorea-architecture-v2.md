# HelloFromKorea.com — Site Architecture v2.0

## Brand Identity

**한줄 정의**: 상황별로 큐레이션해주는 솔직한 로컬 가이드
**슬로건안**: 관광 가이드가 안 알려주는 진짜 한국

**톤 3원칙**:
1. 솔직함 — 별로면 별로. 비싸면 비싸다고. 줄 길면 길다고.
2. 재치 — 정보 전달인데 읽는 맛이 있음. 한줄평이 기억에 남음.
3. 실용 — 예쁜 사진 없어도 진짜 도움이 되는 정보.

**톤 예시 (한국어)**:
- "여기 웨이팅 40분인데 그만큼의 가치가 있냐고? 있음. 근데 비 오는 날 가면 15분컷."
- "인스타에서 본 그 카페 맞음. 근데 2층 자리 아니면 의미 없음."
- "부모님 모시고 갔는데 아버지가 또 가자고 하면 그게 진짜 맛집."

**톤 예시 (영어)**:
- "40-minute wait. Worth it? Yes. Pro tip: go when it rains — 15 min."
- "That Instagram cafe? Only the 2nd floor seats have the view. Otherwise, skip."
- "If a Korean dad says he wants to come back, that's the real Michelin star."

## 언어 전략

**한국어 = 메인 (트래픽 근간)**
- 한국인 검색량이 압도적으로 큼
- 네이버 + 구글 한국어 검색 동시 공략
- 모든 콘텐츠 한국어 우선 작성

**영어 = 확장 (RPM 부스터)**
- 한국어 콘텐츠 중 외국인에게도 유용한 것만 선별 번역
- 영어 트래픽 = RPM 5~10배 → 수익 핵심
- 단, 번역체가 아닌 톤 재해석 (같은 정보, 다른 톤)

**일본어/중국어 = 추후 확장**
- 트래픽 검증 후 고성과 콘텐츠만 번역
- Phase 3 이후 고려

## 핵심 페이지 매트릭스 (상황 × 지역)

이게 pSEO의 진짜 파워. 상황과 지역을 조합하면 자동으로 수백 개 페이지가 나옴.

| | 서울 성수 | 서울 홍대 | 서울 강남 | 부산 해운대 | 제주 ... |
|---|---|---|---|---|---|
| 비오는날 | ✅ | ✅ | ✅ | ✅ | ✅ |
| 혼자갈때 | ✅ | ✅ | ✅ | ✅ | ✅ |
| 데이트 | ✅ | ✅ | ✅ | ✅ | ✅ |
| 부모님과 | ✅ | ✅ | ✅ | ✅ | ✅ |
| 가성비 | ✅ | ✅ | ✅ | ✅ | ✅ |
| 야간 | ✅ | ✅ | ✅ | ✅ | ✅ |

→ 상황 10개 × 지역 50개 = **500페이지 자동 생성 가능**
→ 각 페이지 URL 예시: /상황별/데이트/서울-성수동/
→ 영어: /en/situations/date/seoul-seongsu/

## 키워드 전략

### 키워드 수집 자동화 방법

1. **네이버 자동완성 API 스크래핑**
   - "성수동" 입력 → 성수동 카페, 성수동 맛집, 성수동 데이트 등 자동 수집
   - "비오는날" 입력 → 비오는날 데이트, 비오는날 실내, 비오는날 카페 등
   - 지역 × 상황 조합 키워드 자동 생성

2. **구글 키워드 플래너**
   - 월간 검색량 확인
   - 경쟁도 낮은 롱테일 키워드 우선 공략

3. **네이버 데이터랩 / 구글 트렌드**
   - 시즌별 급상승 키워드 자동 감지
   - 예: 3월 "벚꽃 명소", 10월 "단풍 드라이브", 12월 "크리스마스 데이트"

4. **경쟁 사이트 키워드 역분석**
   - 블루리본, KKday, 다이닝코드 등의 상위 노출 키워드 수집
   - 그 중 우리가 상황별로 더 잘 답할 수 있는 것만 선별

### 키워드 유형별 페이지 매핑

| 키워드 유형 | 예시 | 매핑 페이지 | 월 검색량 추정 |
|---|---|---|---|
| 지역+상황 | 홍대 데이트 코스 | /상황별/데이트/서울-홍대/ | 높음 |
| 지역+카테고리 | 성수동 카페 추천 | /지역/서울/성수동/ | 매우 높음 |
| 상황+조건 | 비오는날 실내 데이트 | /상황별/비오는날/ | 높음 |
| 시즌+지역 | 2026 벚꽃 명소 서울 | /상황별/시즌/벚꽃/ (자동생성) | 시즌 폭발 |
| 솔직 리뷰형 | 성수동 과대평가 카페 | /솔직리뷰/과대평가-주의보/ | 중간 (화제성 높음) |
| 실용 정보 | 제주도 2박3일 코스 | /지역/제주/ | 매우 높음 |

### 우선 공략 키워드 (Phase 1)

검색량 높고 + 경쟁 비교적 낮은 조합:
- [동네명] 맛집 / 카페 / 데이트 (성수, 을지로, 연남, 한남 등 핫플 동네)
- [상황] + 추천 (혼밥 맛집 추천, 데이트 코스 추천 등)
- [시즌] + [지역] (2026 벚꽃 서울, 겨울 강릉 여행 등)
- [솔직 리뷰] 유형 (과대평가 맛집, 진짜 맛집 등 — SNS 바이럴 노림)

## 콘텐츠 자동화 파이프라인

### Tier 1: 상황×지역 자동 생성 페이지 (80%)

**데이터 수집**:
- Google Places API → 장소명, 평점, 리뷰 수, 카테고리, 영업시간
- 네이버 플레이스 → 한국어 리뷰, 방문자 리뷰, 블로그 리뷰 수
- 기상청 API → 날씨 (비오는날 카테고리 연동)
- 공공데이터 → 축제, 행사 정보

**AI 가공**:
- 수집된 데이터를 HelloFromKorea 톤으로 재작성
- 각 장소에 솔직한 한줄평 자동 생성
- "이런 사람에게 추천 / 이런 사람은 패스" 자동 분류
- 가격대, 웨이팅 시간, 꿀팁 구조화

**수빈 투입**: 없음 (자동)
**발행 빈도**: 일 5~10페이지

### Tier 2: AI 큐레이션 리스트 (15%)

**형식**: "○○ BEST 7", "솔직 비교", "상황별 추천"
**데이터**: Tier 1 페이지 데이터 종합 + 트렌드 키워드
**AI 가공**: 리스트 초안 작성 + 톤 적용
**수빈 투입**: 전체 흐름 감수, 한줄평 추가/수정 (편당 10~15분)
**발행 빈도**: 주 3~5편

### Tier 3: 솔직리뷰 (5%)

**형식**: 수빈이 직접 가본 곳 리뷰
**카테고리**:
- 과대평가 주의보 — 유명한데 솔직히 별로인 곳 (화제성 폭발)
- 숨은 보석 — 아무도 모르는데 개좋은 곳
- 재방문각 — 돈 내고 또 갈 곳
**수빈 투입**: 사진 + 메모 → AI가 초안 → 수빈 감수
**발행 빈도**: 월 2~4편

## 수익화 로드맵

### Phase 1 (0~3개월): 기반 구축
- 도메인 등록 + Next.js 사이트 구축
- 핵심 30~50페이지 발행
- Google Search Console + Analytics 설정
- AdSense 승인 신청 (최소 10~15개 양질 페이지 필요)

### Phase 2 (3~6개월): 트래픽 확보
- pSEO로 200~500페이지 확장
- 네이버 서치어드바이저 등록
- SNS 연동 (인스타그램 — 솔직리뷰 티저)
- 월 50,000 PV 목표

### Phase 3 (6~12개월): 수익 최적화
- Mediavine 또는 Ezoic 전환 (50K 세션 달성 시)
- 영어 버전 확장 (고RPM 트래픽 확보)
- 일본어/중국어 확장 검토
- 애필리에이트 추가 (야놀자, 클룩, 마이리얼트립 등)
- 월 300만원 수익 목표

### Phase 4 (12개월~): 스케일업
- 1,000페이지 이상
- 뉴스레터 + 커뮤니티 확장
- 브랜드 협업/광고 수익 추가

## 기술 스택

| 구성 | 도구 | 이유 |
|------|------|------|
| 프레임워크 | Next.js 14+ (App Router) | SSG로 빠른 속도 + pSEO 최적화 |
| 호스팅 | Vercel | Next.js 최적, 무료 시작 가능 |
| CMS | MDX 또는 Contentlayer | 마크다운 기반, 개발자 없이 글 작성 가능 |
| 데이터베이스 | Supabase (무료 티어) | 장소 데이터, 리뷰 데이터 저장 |
| 자동화 | Claude API + Cron Job | 콘텐츠 자동 생성 파이프라인 |
| 이미지 | Unsplash API + 자체 촬영 | 저작권 안전 + 실제 사진 |
| 분석 | Google Analytics 4 + Search Console | 트래픽/키워드 분석 |
| 광고 | AdSense → Mediavine | 단계별 전환 |
| 도메인 | Namecheap 또는 Cloudflare | 저렴 + 관리 편리 |
